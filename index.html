GIT版本控制工具的学习

NO.one

  1. 初始化：git init
  2. 添加：git add .
  3. 提交：git commit -m "message"
  4. 查看日志：git log
  5. 只查看日志文本：git log --oneline
  6. 查看状态：git status
  7. 清屏：clear
  8. 修改用户：git config --global user.email "xxx@qq.com"
  9. 修改用户：git config --global user.name "xxx"

NO.two

  1. 回滚：git reset --hard head~1 (回滚到上一个版本)
  2. 查看所有的修改日志：git reflog
  3. 回滚：git reset --hard 版本号(回滚到某个版本)

NO.3 分支管理

  1. 创建分支：git branch 分支名
  2. 切换分支：git checkout 分支名
  3. 创建并切换分支：git checkout -b 分支名
  4. 查看分支：git branch
  5. 合并分支：git merge 分支名(先切换回master分支)
  6. 删除分支：git branch -d 分支名

NO.4

  1. 忽略：.gitignore

NO.5

  1. 本地项目推到远程仓库：git push url master
  2. 定义变量：git remote add 变量名 https://github.com/LIJING0906/GITstudy.git
  3. 删除变量：git remote remove 变量名
  4. 关联远程仓库：git push 变量名 master

NO.6 密钥

  1. 生成密钥：ssh-keygen -t rsa(在任何位置git bash)

NO.7 克隆和拉取

  1. 克隆：git clone 地址
  2. 拉取：git pull 地址
